Plugin e Template de Consulta de Clima para WordPress Visão Geral Este projeto combina um plugin e um template para WordPress que obtêm dados meteorológicos atuais e uma previsão de 5 dias usando a API OpenWeatherMap. O plugin exibe a temperatura, a sensação térmica e uma breve descrição das condições meteorológicas, enquanto o template proporciona uma interface visual atrativa para exibir essas informações.
